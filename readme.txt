# Global Patent Intelligence Dashboard

This repository contains a high-performance data pipeline and an interactive Streamlit dashboard for analyzing global patent distribution, inventor trends, and company intelligence using USPTO PatentsView data.

## Project Links

-  Dashboard Links
 Local URL: http://localhost:8501

- GitHub Repository: https://github.com/NaluyindaMoureen01/Data_pipeline

## How to Run

1. Install the required Dependencies:

   install streamlit pandas plotly sqlite3
   

2. Run the Dashboard:
   
   streamlit run scripts/dashboard.py

## Features
- Interactive Choropleth Map (Click-to-filter by country)
- Real-time KPI metrics
- Company and Inventor ranking lists
- Time-series patent filing trends
- Full data tables for deep inspection
